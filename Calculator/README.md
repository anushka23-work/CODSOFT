# Simple Python Calculator

A command-line based simple calculator application developed in Python that performs basic arithmetic operations. This project showcases clean code structure using functions, input validation, and error handling.

## Features
- **Basic Operations:** Addition, Subtraction, Multiplication, and Division.
- **Error Handling:** Safe division handling (prevents division by zero error) and input validation (prevents crashes on non-numeric inputs).
- **Continuous Execution:** Runs in a loop until the user explicitly chooses to exit.

## How to Run
1. Make sure you have Python installed on your system.
2. Run the following command in your terminal/command prompt:
   ```bash
   python calculator.py
