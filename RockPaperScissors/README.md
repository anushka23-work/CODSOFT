# Rock-Paper-Scissors Game

A classic interactive game developed in Python, where the user competes against the computer. This project features a user-friendly GUI, real-time score tracking, and intuitive feedback mechanisms.

## Features
- **Interactive GUI:** Built using `tkinter` for a smooth visual experience.
- **Score Tracking:** Keeps a live count of both the user's and the computer's scores.
- **Color-Coded Feedback:** Dynamic visual cues (Green for Win, Red for Lose, Yellow for Tie) to provide immediate game status.
- **Game Control:** Includes a 'Reset' feature to clear scores and start a new session instantly.

## How to Run
1. Make sure you have Python installed on your system.
2. Navigate to the project directory in your terminal/command prompt.
3. Run the following command:
   ```bash
   python rock_paper_scissors.py
